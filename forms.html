Input controls are the HTML input elements:

  • input elements
  • select elements
  • button elements
  • textarea elements

The novalidate of form attribute is new in HTML5. It disables any default browser validation.

The novalidate attribute is not needed for this application, but normally you will use it in AngularJS forms, to override standard HTML5 validation.
